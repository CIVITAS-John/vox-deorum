# v0.8.0 — 2026-03-03

## Oracle Mode (New)
- Introduced standalone **Oracle mode** — a conversational AI interface for asking questions about the game state outside of the turn cycle
- Full Oracle pipeline: agent, prompt extractor, model resolver, DB resolver
- Multiple system prompt support for flexible Oracle interactions
- Iterative refinements to Oracle reliability and timeout handling

## Vox Agents
- **Model support**: Added Qwen 3.5 and hosted MiniMax-M2.5 middleware
- Updated model roster and concurrency limits
- Improved tool-rescue middleware and guardrails
- Sends model info and action records to the UI for observer display

## UI
- New **Model Options dialog** for fine-grained model configuration
- Improved config view layout and styling
- Enhanced config utility helpers

## MCP Server
